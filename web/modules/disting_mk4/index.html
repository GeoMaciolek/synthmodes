Info

Disting mk4

Disting mk4 is a eurorack swift army knife. It is made by Expert Sleepers.

The Disting firmware gets updated regularly, be sure to check the firmware page for the latest version and latest manual.

These cheat sheets cover firmware V4.21.

To change algorithms, press S once to enter the menu, and a second time to select the first menu item.

Turn the S encoder to change the current selected parameter. Press Z on algorithms that do not have knob recording, or press and turn S to change which parameter is selected.

For additional details about parameters, scales, and note numbers see the excellent cheatsheet by vo1t.

Bit Reduction Modes

There are 2 bit reduction modes.
Type I: sample treated as 16 bit values, low bits thrown away. Each step in bit reduction halves the resolution.
Type II: sample is divided by a factor. Allows for a smoother reduction of resolution.
Parameter 1 controls which bit reduction type applies to negative and positive values.
