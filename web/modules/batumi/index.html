Batumi

Batumi is a versatile 10HP quad LFO module with voltage controlled parameters. It can operate as four independent LFOs, or in one of three sync modes. Its frequency range is from a super low 28 hours in voltage controlled divide mode, up to 500Hz.

The four sliding pots control different parameters depending on the current mode. LEDs are used to indicate the current mode. The button in the centre of the module is used to change the current mode.
