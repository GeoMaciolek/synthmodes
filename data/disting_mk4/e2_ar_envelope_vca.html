Modes 3, 4, 5 act like modes 0, 1, 2 except the trigger is derived from the envelope of input X. Modes 6, 7, 8 act like modes 0, 1, 2 except the VCA input drives the envelope of input Y.

0 0 8 0
Trigger Mode.
When 0, acts as AR envelope, holding envelope high as long as trigger is high.
When 1, executes one full A/D when gate goes high.
When 2, loops A/D as long as the trigger is high.

1 0 1 0
Z mode.
When 0, Z sweeps from short A&D -> short A, long D -> long A&D -> long A, short D
When 1, sets A&D to same value.
