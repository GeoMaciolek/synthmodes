Front panel controls

Track buttons:

  • If in live mode, the buttons are individual Track 1 and 2 MUTE buttons.

  • If in step record mode, the buttons are used for adding steps and rests.

  • If in Euclidean record mode, the buttons set the length of the Euclidean circle (+1 step pressing STEP and +16 steps by pressing REST).

External Clock input: 16th notes clock expected (PPQN24÷6) (0...+5V or +10V, 0.65V threshold).

Rate/divider knob:
  • If no clock cable is inserted, then this knob sets the speed of the internal clock in BPM, from 30 to 300.

  • If the clock is inserted via cable, this knob sets the clock division / multiplication of the pattern.

  • If the Rate knob is set to Tuplets (fully CCW), this activates a special mode where the Repeats for each track are now stretched within a beat, which lets you create triplets, quadruplets and other interesting and uncommon rhythms in eurorack.

Repeats 1/2: manual knobs with CV inputs. Set amount of active trigger repeats or Euclidean trigger fills. Additionally these are used to set internal pattern modifiers..

Repeats 1/2 CV inputs: When a CV cable is inserted, the last position of the Repeats knob is remembered and each Repeats knob acts as a CV attenuator; expected CV range: 0...+5V. A simple formula for the Repeats parameter value with the cable inserted is:

  • Repeats value = Last stored value + (CV input value x Attenuator amount).

Reset/synchronisation trigger inputs: each trigger applied resets a certain track’s sequence to its first step. ‘Reset in 1’ is normalled to ‘Reset in 2’, if no cable is inserted into ‘reset in 2’

Clock output: internal clock out jack: 30...300 BPM in 16th notes (PPQN24÷6) when no external clock applied; external clock output in 16th notes after divider; output range is 0...+10V.

Track selector switch: record a trigger sequence or set the Euclidean length by switching to the left (track 1) or right (track 2); playback live mode in the middle.

Track 1 and 2 trigger outputs:: trigger output length of 10 msec with output range of 0...+10V.

LED indicator: blinking to BPM in blue in live mode; lights up red in step recording; fuchsia in euclidean recording.
