Overview

Running order consists of 2 identical and independent trigger tracks that are clocked by ultra stable zero-jitter internal clock or by using external clock input, that can be additionally processed by the onboard divider / multiplier knob. Additionally, each track features a reset input for further synchronisation flexibility, mute button for performance control and 8 pattern slots that can be stored and recalled later for each of the tracks.

There are 2 main pattern types:

  • 101 style 1/16th note input mode (step/rest) with manual or CV controllable amount of trigger repeats when the active trigger step happens, internal per step trigger probability and ratcheting.
  • Euclidean, where we set Euclidean total number of steps in the circle and amount of triggers happening in that circle. The pattern modifiers include euclidean pattern rotation either internally or cv controlled and internal global trigger probability of the pattern.
