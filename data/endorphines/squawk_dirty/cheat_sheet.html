Cheat Sheet for things you might forget:

  • To switch between 8 different filter type press the Mode button.


  • The first 4 filters are shown using fully lit LEDS, filters 5-8 are shown by dimply lit LEDs.


  • The following filter types are available in order 1-8: 24DB/OCT TRANSISTOR LADDER, MS-20 12DB/OCT, 12DB/OCT LOW PASS GATE, 12DB/OCT RESONANT LOW PASS GATE, 12DB/OCT DJ ISOLATOR BIPOLAR VARIABLE FILTER, 12DB/OCT HIGH PASS STATE VARIABLE FILTER, 12DB/OCT BAND PASS STATE VARIABLE FILTER, COMB FILTER / RESONATOR.


  • To activate Meta Scanning of filter types hold press the Mode button for 1 second, apply external CV to Resonance CV input with Resonance knob acting as attenuator.


  • To activate All Access Pass mode hold press the Mode button for longer than 2.5 seconds, the Illuminati LED will light up Fuchsia. Use the HPF knob to adjust the delay on Output 2.
