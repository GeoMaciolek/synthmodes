Front panel controls

Mode Button: pressing the MODE button shortly cycles thru all filter types. Long MODE press enables meta filter scanning. Extra long MODE press enables the all-pass mode.

VCA CV in: unattenuated CV input for VCA (post-filter): range 0...+5v .

High Pass Filter knob: manual control of the High Pass Filter and attenuator for CV for high-pass filter once the cable is inserted to HPF CV in. HPF is placed before the main multimode filter.

HPF CV in: cv control of the HPF, range 0…+5V.

VCA CV input: Unattenuated CV input for volume control with range 0….+5V.

Cutoff Knob: manual control of the Filter Cutoff, once the cable is inserted to the CV input it acts as attenuator.

Cutoff CV in: cv control of the Filter Cutoff, range 0…+5V.

Resonance knob: manual control of the emphasis (resonance peak) of the frequency, in some other cases it controls the bandwidth, vactrol’s decay etc. Acts as attenuator when a patch cable is inserted into the Resonance CV input.

Resonance CV input: cv control of the Resonance parameter, range 0…+5V.

Audio inputs: audio inputs for the filter, if nothing is plugged into input 2, input 1 is internally connected to input 2.

LEDs: 4 white LEDs show the currently selected filter type. When the LED is fully LIT, it shows one of I...IV chosen filter types. When the LED is semi LIT, it shows one of V...IV chosen filter types. Status LED in the middle is used to show the update process.

Audio outputs: final post filter audio outputs.
